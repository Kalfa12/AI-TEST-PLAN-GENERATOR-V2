# Exports

Generated plans can be exported for review or presentation.

## PDF Export

The frontend contains PDF export logic for test plan details.

Good exported reports should include:

- project name;
- plan title;
- scope;
- risks;
- test cases;
- acceptance criteria;
- traceability;
- coverage information.

## Export Advice

Export only after generation has completed and the plan detail page looks correct.

If the export has visual layout issues, show the app view instead and keep the PDF as a backup artifact.
