# AI Test Plan Generator

AI Test Plan Generator is a web platform for transforming technical documents into structured, traceable test plans.

The application is designed for engineering and QA teams that need to:

- upload specifications and technical documents;
- extract requirements from those documents;
- generate test plans and test cases with AI agents;
- keep traceability between source documents, requirements, tests, and coverage;
- manage reusable knowledge and project-specific knowledge;
- use a contextual chatbot during analysis and review;
- export and present test results in a professional format.

This documentation explains the product, the architecture, the AI workflow, how to install the system, and how to demonstrate it.
